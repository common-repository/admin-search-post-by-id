Search and filter your posts by ID in the admin section.

== Description ==

Admin Search Post By ID add a new field in the search, filter your posts by ID and add a new column to display the post ID in the admin post listings.

It works for Posts and Custom Post Types.

== Installation ==

1. Upload `admin-search-post-by-id` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
